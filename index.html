Desarrollo de ETL

EEste proyecto consistió en el desarrollo de una base de datos relacional en SQL Server a partir de diversas tablas existentes. Se empezó por diseñar y crear las tablas necesarias mediante scripts SQL, especificando tipos de datos y restricciones para garantizar la integridad y consistencia de la información. Luego, se elaboraron procedimientos almacenados (stored procedures) para los procesos de ETL (Extract, Transform, Load), cuyo objetivo era extraer datos de las tablas originales, transformarlos según los requisitos del nuevo esquema y cargarlos en las nuevas tablas relacionales. Durante la transformación, se verificó la no duplicación de datos y se realizó el casteo correcto de los campos para asegurar que los datos se almacenaran en el formato adecuado. Además, se establecieron las relaciones entre las tablas utilizando claves foráneas, lo que garantiza la correcta vinculación y la integridad referencial de la base de datos.

SQL + Python

Este proyecto consistió en la migración de una base de datos compuesta por tablas de Excel a una base de datos relacional en SQL. El proceso incluyó la limpieza exhaustiva de datos para eliminar duplicados, corregir inconsistencias y asegurar la calidad de la información. Posteriormente, se propusieron y ejecutaron modificaciones en la estructura de la base de datos para optimizar su eficiencia y escalabilidad. Además, se generó un prototipo de buscador en Python, el cual se integró con la nueva base de datos relacional, permitiendo realizar consultas eficientes y mejorando significativamente el acceso y la usabilidad de la información almacenada.

IGALL-CRUD

IGALL CRUD es un proyecto de back-end basado en la base de datos generada en el proyecto de SQL Server. Desarrollado en Python, esta propuesta tiene como objetivo mostrar al cliente las ventajas de migrar a una base de datos robusta y facilitar los procesos de consulta, carga y modificación de datos.
